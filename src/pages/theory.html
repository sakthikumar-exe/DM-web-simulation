Delta modulation is a digital signal encoding technique that converts analog signals into digital data by transmitting only one bit per sample, representing whether the input amplitude has increased or decreased from the previous step. The modulator employs a staircase approximation that tries to follow the shape of the input analog signal using discrete step increments and decrements, based on the comparison with the previous sample. This staircase signal is generated with a fixed step size, Δ, resulting in the output being confined to two levels (+Δ or -Δ) for each sampling instant.

Modulator Working:

● The incoming signal enters the first summing block, where it is compared with the system’s own internally generated staircase value.

● The summing block subtracts the staircase value from the input, producing an error signal. This error simply tells whether the input is higher or lower than the staircase.

● This error is then fed into a 1-bit quantizer. The quantizer checks the sign of the error. If the error is positive, the quantizer outputs a “1", else it outputs "0"

● The quantizer output is fed into the accumulator. The accumulator adds a fixed step (Δ) either positively or negatively depending on whether the bit was “1” or “0".

● This updated value then goes through a one-sample delay block to store this staircase value until the next sample time, so it can be used for the next comparison.

● The new staircase value is fed back to the first summing block. This closes the loop.

● On the next sampling instant, the new staircase is compared again with the new input sample.

● The quantiser output which is the Delta Modulated Bitstream is then encoded and transmitted.

● This up-and-down stepping forms the characteristic staircase approximation of the original signal.

Demodulator Working:

● The function of the Demodulator's Accumulator is same as the Modulator's Accumulator

● The Delta Modulated Bitstream is fed into the accumulator. The accumulator adds a fixed step (Δ) depending on the bit

● This forms the Staircase approximation of the original signal

● This staircase approximation is fed into a Reconstruction Filter (typically a LPF) to reconstruct the original signal from the staircase approximation

● The LPF removes high frequency quantization noise and smooths step transition


Noises in Delta Modulation

Figure 1: Errors in Staircase Approximation



In Delta Modulation, because the system works with only one bit and a fixed step size, it cannot perfectly follow every kind of input waveform. As the input signal moves, the staircase that tries to follow it may sometimes move too aggressively or too slowly. These mismatches create two characteristic types of distortion: granular noise and slope overload distortion.

Granular Noise or Quantizing Noise:

Granular noise appears when the input signal is almost flat or changes very slowly, but the staircase continues to jump up or down by the same fixed amount every sample. Since the staircase is forced to move by the step size even when the input hardly changes, it ends up wobbling around the true value instead of settling smoothly. This jittering up-and-down movement creates a grainy or noisy appearance in the reconstructed signal. It is called as Granular noise since the output looks rough and coarse, especially when the input is gently varying.

Slope overload distortion:

Slope overload distortion occurs when the input signal changes too quickly. The staircase in DM can only move a fixed amount (Δ) per sample, so if the input makes a rapid rise or fall, the staircase simply cannot climb or descend fast enough to keep up. The result is that the staircase falls behind the true signal, and the gap between them grows over several samples. Instead of following the signal closely, the reconstructed waveform looks flattened or delayed during sharp transitions.

Both of these noises come from the limitation of using a single, fixed step size. If the step is too large, the staircase will overshoot and jitter during slow signal changes, producing granular noise. If the step is too small, the staircase won’t be able to chase fast-changing sections, resulting in slope overload. DM therefore has an inherent trade-off: choosing a step size that balances both types of error
