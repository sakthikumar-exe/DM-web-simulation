Working of Practical Circuit

Figure 1: Delta Modulation Circuit


Circuit Overview

(a) Comparator:

The comparator compares the instantaneous amplitude of the input message with the integrator output and generates a High or low output depending on whether the input signal is greater or smaller than the integrator output.

(b) Digital Sampler:

The IC 4O13 is a dual D-type flip flop. The flip flop simply holds or remembers its previous output state.
The Flip Flop changes its output value to its current input when a clock pulse is applied at CLK

The comnparator output is given as input to the Flip Flop. The Flip Flop samples the Comparator output at the sampling frequency given as CLK input.
The output of the Flip Flop is the Delta Modulated Bitstream

(c) Integrator:

The integrator receives the sampled digital output from the flip flop and produces an analog output proportional to the sum of all previous inputs. A resistor and a capacitor determine the integration time constant (RC). The output voltage of the integrator at any time is given by:

Vout(t) = − 1 RC ∫ Vin(t) dt

The integrator reconstructs the signal based on the RC time constant. The RC time constant determines the rate at which the integrator output changes in response to the digital input. A properly chosen RC value allows the integrator to accurately track the input signal while minimizing distortion. If the RC time constant is too large, the integrator responds slowly, leading to slope overload distortion for rapidly changing signals. Conversely, a very small RC value causes the integrator output to fluctuate excessively, resulting in granular noise. Hence, the RC time constant is selected as a compromise between tracking ability and noise performance in Delta Modulation systems.

(d) lnverting amplifier:

The inverting amplifier provide an inverted version of the integrator output. The resistor pair ensures unity gain inversion, The inverted signal is given as a feedback to the comparator.

Step-by-Step Working:

1. Message signal is given as an input to the comparator

2. Comparator compares message with inverted feedback from the integrator.

3. Comparator output is sampled by the flipflop to generate DM output

4. DM Output drives Integrator:

• If DM output is 1 --> Integrator output steps up.

• If DM output is 0 --> Integrator output steps down.

5. Integrator output is sent to Inverter which feeds back inverted signal to the Comparator.

6. The process repeats continuously.
