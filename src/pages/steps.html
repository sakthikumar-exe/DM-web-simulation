Steps to Run the Simulation

Message Signal:

Specify the amplitude and frequency of the message signal.

Sampling Pulse:

Specify the frequency of the sampling pulse. This determines the time interval at which the message signal is sampled and the delta-modulated bitstream is generated.
Ensure that it satisfies the Nyquist Theorem (Fs â‰¥ 2Fm).

RC Time Constant:

The RC time constant specifies the rate at which the capacitor charges and discharges in the integrator.
A lower RC time constant results in faster capacitor charging.
Specify the value such that the integrator output follows the original signal with minimal noise.
In simple terms, this value determines the step size.

The AutoSet switch automatically sets the RC time constant such that the integrator output follows the message signal with minimal noise.

Demodulator:

No input parameters are required for the demodulator block.
The accumulator is an integrator whose function is the same as that used in the modulator.
The reconstruction filter is a low-pass filter that removes high-frequency noise and smooths the accumulator output, thereby reconstructing the message signal.
(A third-order IIR Butterworth LPF is used in this simulation.)

Output Graphs:

The Staircase Approximation switch toggles the display of the staircase approximation plot over the message signal, allowing the study of the effect of step size on noise.

The view of the plot can be modified using the tools available at the top-right corner of the plot.

Note: There will be a non-linear phase distortion (delay) in the reconstruction filter output with respect to the accumulator output. This delay is due to the causal nature of the Butterworth LPF. The present output of a Butterworth filter depends on the present input, past inputs, and past outputs. This introduces delay elements, which result in the observed delay.
